# Installation

To install the package, you can use the following command:

```bash
pip install sysext-creator
```

# Usage

Here's how you can use the sysext-creator:

```python
from sysext_creator import create

create(parameters)
```

# Automation

To automate this process, you can include this in your scripts:

```bash
#!/bin/bash
sysext-creator --option=value
```

Make sure to adjust the options as needed.
